# WhatsApp-шлюз (Baileys)

Отдельный Node-сервис для платформы рассылки: вход в WhatsApp по QR-коду и API отправки сообщений.

## Запуск

```bash
cd whatsapp-gateway
npm install
npm start
```

Затем открой **http://localhost:8667** в браузере и отсканируй QR-код:
WhatsApp на телефоне → **Настройки → Связанные устройства → Привязка устройства** → наведи камеру на QR.

Сессия сохраняется в папке `auth/` — сканировать нужно **один раз**. Дальше сервис
подключается сам.

## API

| Метод | Путь | Назначение |
|------|------|-----------|
| `GET` | `/` | Страница с QR-кодом и статусом |
| `GET` | `/status` | `{ connected, hasQr, user }` |
| `GET` | `/qr` | `{ qr: <data-url или null> }` |
| `POST` | `/send` | тело `{ phone, message }` — отправить текст |
| `POST` | `/logout` | сбросить сессию (новый QR) |

Пример отправки:

```bash
curl -X POST http://localhost:8667/send \
  -H "Content-Type: application/json" \
  -d '{"phone":"77001112233","message":"Здравствуйте!"}'
```

## Важно

- Массовые рассылки против правил WhatsApp — номер могут заблокировать.
  Используй отдельную SIM, небольшие объёмы и задержки (антибан-логику добавляет
  Python-оркестратор кампаний).
- Порт `8667` должен совпадать с `OutreachOptions.gateway_url` в конфиге Python.
